# Docling: Getting Started

## Overview

Docling is a document conversion and processing framework. The getting started page serves as an entry point to the platform, offering guidance through several key areas.

## Main Navigation Sections

The documentation is organized into several primary categories:

- **Installation**: Setup instructions for the Docling environment
- **Usage**: Foundational guidance on basic operations
- **Concepts**: Core principles underlying the framework
- **Examples**: Practical implementations across various use cases
- **Integrations**: Connections with popular AI tools and frameworks
- **Reference**: Detailed API documentation

## Example Categories Available

The platform includes comprehensive examples organized by function:

- Conversion workflows (simple, custom, batch, and multi-format)
- Serialization and chunking operations
- Information extraction tasks
- RAG (Retrieval-Augmented Generation) implementations
- Picture annotation capabilities
- GPU optimization techniques

## Integration Ecosystem

Docling integrates with numerous frameworks including:

- AI development platforms (LangChain, LlamaIndex, Haystack)
- Specialized tools (spaCy, Prodigy, NVIDIA)
- Vector databases (Qdrant, Weaviate, Milvus)
- Enterprise solutions (Azure, OpenSearch, MongoDB)

## Community Resources

The project maintains active community channels on GitHub, Discord, and LinkedIn for support and engagement.
