# Docling Examples

## Overview

The Examples section provides numerous application recipes and end-to-end workflows for Docling, organized into several categories:

### Featured Starting Points

**Conversion Examples**
"ranging from simple conversion to Markdown and export of figures & tables, to VLM and audio pipelines"

**RAG Implementations**
"various RAG examples, e.g. based on LangChain, LlamaIndex, or Haystack, including visual grounding, and using different vector stores"

**Enrichment & Annotation**
- Picture annotations and enrichments
- Visual grounding capabilities
